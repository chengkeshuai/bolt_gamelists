## 开发环境

- Node.js
- React + TypeScript
- Vite
- Tailwind CSS

## 如何运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 访问网站：
```
http://localhost:4000
```

## 项目结构

```
src/
  ├── components/     # React组件
  ├── context/       # React上下文
  ├── styles/        # 样式文件
  └── public/        # 静态资源
      └── games/     # 游戏文件
```
